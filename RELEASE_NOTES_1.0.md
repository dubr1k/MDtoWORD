# MDtoWORD 1.0

## Русский

Первый полноценный настольный релиз MDtoWORD — конвертера Markdown ↔ Microsoft Word с графическим интерфейсом.

### Возможности

- Конвертация Markdown в DOCX и Word в Markdown.
- Пакетная обработка нескольких файлов.
- Настройка шрифта и размера текста.
- Русский и английский интерфейс.
- Автономные приложения: Python и необходимые библиотеки уже включены.
- Новая адаптированная иконка приложения.

### Загрузки

- **macOS 12+ / Apple Silicon:** `MDtoWORD-macOS-arm64.zip`
- **Windows 10/11 / x64:** `MDtoWORD-Windows-x64.zip`

Распакуйте архив и запустите приложение. В macOS при первом запуске может потребоваться нажать правой кнопкой на приложении и выбрать «Открыть». Windows SmartScreen также может показать предупреждение, поскольку сборки пока не подписаны коммерческими сертификатами.

## English

The first complete desktop release of MDtoWORD — a graphical Markdown ↔ Microsoft Word converter.

### Features

- Convert Markdown to DOCX and Word documents to Markdown.
- Batch processing for multiple files.
- Configurable font family and text size.
- Russian and English user interface.
- Self-contained applications with Python and required libraries included.
- A new platform-friendly application icon.

### Downloads

- **macOS 12+ / Apple Silicon:** `MDtoWORD-macOS-arm64.zip`
- **Windows 10/11 / x64:** `MDtoWORD-Windows-x64.zip`

Extract the archive and launch the application. On macOS, the first launch may require right-clicking the app and selecting “Open”. Windows SmartScreen may also display a warning because the builds are not yet signed with commercial certificates.
